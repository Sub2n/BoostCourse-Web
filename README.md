# BoostCourse-Web

부스트코스 웹 프로그래밍

## 1. 자기소개 페이지 만들기

### 웹 프론트엔드 기술 요구사항

- html layout tag 사용
- classname 일정한 컨벤션 유지
- semantic tag 사용 (div 사용 최대한 자제)
- position과 float로 element 배치
- library X
- id와 class 등 다양한 selector 문법 활용

### 웹 백엔드 기술 요구사항

- 톰캣 서버를 통해서 자기소개 페이지가 동작돼야 함
- 서블릿 페이지 하나를 생성해서 url을 입력했을 때 시간데이터가 화면에 노출돼야 함 (디자인 x)
